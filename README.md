# ✨ Intellis AI – Stealth Meeting Assistant

**Intellis AI** is a real‑time AI assistant that runs as an invisible overlay during screen‑shared meetings.  
It answers questions via text, voice, or screen capture — and **stays hidden** from Zoom, Google Meet, and Teams.

![License](https://img.shields.io/badge/License-MIT-green.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-blueviolet.svg)

---

## 🚀 Features

- 💬 **AI Q&A** – Type any question, get instant answers
- 📸 **Full Screen Capture** – Analyze anything on screen
- 📐 **Region Capture** – Select specific area
- 🎤 **Voice Input** – Auto‑detect questions from audio
- 👻 **Stealth Mode** – Dim overlay during meetings
- 🔒 **Invisible During Screen Share** – Windows Display Affinity API
- 🧠 **Markdown Support** – Bold, italics, headings, code blocks in responses
- ⚡ **Real‑time** – Responses in 1‑3 seconds
